Hello Again

技术

  1. Jekyll 和 Hexo ，有什么区别。
  2. deploy 很好用啊。
  3. 但是源码和静态文件应该同时同步的话有什么自动化的方法吗。
  4. 搜索对中文的支持是怎样的的。
  5. 新的标签和分类应该怎样添加。
  6. 新建博客自动打开添加 js 自动监听新建事件。
  7. Markdown 文件和有道云笔记。
  8. 博客是否应该分成三栏呢。

域名

  1. 开始觉得阿里的麻烦之处了。
  2. .wang 域名还不是很普及的样子。
  3. 域名简洁点好还是有意义更好。
  4. 自己的名字的域名有没有必要继续持有。
  5. .me 域名是不是显得很小家子气。
  6. .com 域名虽然很美观但是用起来还真是挺麻烦。

托管

  1. 国内的话就是 coding.net 啦。
  2. github 有的时候访问还是有点慢。
  3. 博客作为一个项目应该是更好的方式。
  4. 源码可以放在 master 分支中，静态网页可以放在 gh-pages 分支中。
  5. 独立的空间貌似是没有必要的。
  6. 我甚至其实都根本不写东西。
